Gestionnaire MSIX

Gérez les paquets UWP installés sur votre système Windows (permet entre autre de supprimer le bloat)

Attention : Ce logiciel peut supprimer n'importe quel paquet MSIX/APPX que la commande Remove-Package de PowerShell est capable de supprimer (donc presque tous), son utilisation n'est donc pas recommandée aux utilisateurs ne savant pas ce qu'ils font. Le createur (Luigiday) n'est pas responsable en cas de probleme causés par la mauvaise utilisation de cette application, vous l'utilisez donc a vos risques et périls.

À propos de l'application

Cette application permet de supprimer les paquets d'applications UWP (paquets MSIX) sur Windows 10 et 11. MSIX Manager peut supprimer n'importe quel paquet MSIX/APPX qui est supprimable par PowerShell mais avec une interface plus simple.
